# RailwayTicketReservationSystem
A DBMS-based Railway Ticket Reservation System that efficiently manages passenger details, train schedules, seat availability, and fare calculations using a well-structured relational database.

---

# Project Overview

The Railway Ticket  Reservation System, is a relational database-based application developed to replace manual railway ticket booking systems with a secure, efficient, and user-friendly digital platform.

The system manages:
* Passenger details
* Train schedules
* Seat availability
* Ticket bookings 
* Fare calculations
* Administrative controls

---

# Objectives

* Automate railway ticket booking and management
* Provide real-time seat availability
* Reduce manual errors and data redundancy
* Improve operational efficiency
* Ensure secure authentication and data privacy

---

# System Architecture

The system is divided into the following major modules:

## User Module

* User registration & login
* Search trains
* Check seat availability
* Book tickets
* Cancel tickets
* Manage bookings

## Admin Module

* Add/modify train details
* Manage schedules and routes
* Monitor reservations
* Generate reports
* Maintain database integrity

## Train Management

* Train names and numbers
* Routes and schedules
* Available classes
* Real-time seat updates
